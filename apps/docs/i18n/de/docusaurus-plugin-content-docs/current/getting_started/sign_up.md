# Anmelden

Bereit, mit GOAT zu starten? Folgen Sie diesen einfachen Schritten, um Ihr Konto zu erstellen und die Welt der Geodatenanalyse zu erkunden:


<div class="step">
  <div class="step-number">1</div>
  <div class="content">Rufen Sie die <a href="https://goat.plan4better.de/home">GOAT-Anmeldeseite</a> auf, klicken Sie auf <code>Konto erstellen</code>, füllen Sie Ihre Daten aus und klicken Sie auf <code>Loslegen!</code>, um auf das Registrierungsformular zuzugreifen.</div>
</div>

<div class="step">
  <div class="step-number">2</div>
  <div class="content">Überprüfen Sie Ihr E-Mail-Postfach auf die Bestätigungs-E-Mail von GOAT.</div>
</div>

<div class="step">
  <div class="step-number">3</div>
  <div class="content">Sie sind bereit! Melden Sie sich an und beginnen Sie, GOAT zu nutzen.</div>
</div>

## Fehlerbehebung

### Passwort vergessen?
Wenn Sie Ihr Passwort vergessen haben, klicken Sie auf den Link **Passwort vergessen?** auf der Anmeldeseite. Folgen Sie den Anweisungen, um Ihr Passwort zurückzusetzen.

### Bestätigungs-E-Mail nicht erhalten?
Überprüfen Sie Ihren Spam-Ordner. Falls Sie die E-Mail immer noch nicht finden können, kontaktieren Sie unser Support-Team unter [support@plan4better.de](mailto:support@plan4better.de).

### Benötigen Sie Hilfe?
Falls Sie während des Anmeldeprozesses auf Probleme stoßen, ist unser Support-Team da, um zu helfen. Kontaktieren Sie uns unter [support@plan4better.de](mailto:support@plan4better.de).
